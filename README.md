# Memento

**ECE 49595 - Team 5 | Software Track | Fall 2025**

Memento enables professionals and students to know names and roles before conversations begin, leading to more confident introductions, efficient discussions, and stronger connections at networking events.

## Overview

Memento uses facial recognition on smart glasses to identify people at events and display their profile information (name, role, major, professional experiences) in real-time. The app also calculates similarity scores between users and generates conversation starters based on shared interests.

## Features

- Real-time facial recognition using smart glasses camera
- Profile context display (name, major, department, professional experiences)
- Similarity scoring between users based on shared interests
- Conversation starter generation
- Privacy-focused with user consent and event-based access controls

## Tech Stack

- **Frontend**: React
- **Backend**: FastAPI
- **Database**: Supabase (Auth, RLS, profiles)
- **Facial Recognition**: AWS Rekognition
- **AI**: OpenAI API
- **Hardware**: Mentra Smart Glasses
